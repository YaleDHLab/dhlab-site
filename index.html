Coding with Clarity: Best Practices for Generating Scripts


Monday, November 11, 2024
10-11 a.m. EST
Online

Web scraping in Python? Crunching numbers in R? Make sure you (and others!) can understand and reuse your code! This workshop will teach you some language-agnostic best practices for coding, including writing effective documentation, producing readable code, working towards modular programming, and choosing appropriate data structures.

When You Can’t Find the Data You Need: Advanced Data Collection Methods


Wednesday, November 13, 2024
10-11 a.m. EST
Online

Learn about the benefits and drawbacks of four advanced data collection methods -- manual data collection, web scraping, data requests, and application programming interfaces (APIs) -- at this virtual workshop.
